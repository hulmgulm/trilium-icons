# Icon Packs for TriliumNext Notes

## Emoji Icon Pack

This icon pack allows to use most of the emoji characters as note icons. It is based on Unicode v17.0.
<img width="640" height="649" alt="image" src="https://github.com/user-attachments/assets/272c3b15-0345-4e94-8ed4-863b9492383a" />

It uses the English default emoji descriptions from [this page](https://unicode.org/emoji/charts/full-emoji-list.html). It uses the `emojis` prefix.

See the TriliumNext documentation about [how to install the icon pack](https://docs.triliumnotes.org/user-guide/concepts/themes/icon-packs).

The font included is the [Google Noto Color Emoji](https://github.com/googlefonts/noto-emoji/tree/main/fonts) font v2.051, distributed by Google under the [SIL Open Font License](https://raw.githubusercontent.com/googlefonts/noto-emoji/refs/heads/main/fonts/LICENSE).

In the tools directory, you can find a nodejs script, which was used to create the manifest for this icon pack.

## Font-Awesome Icon Packs

This icon pack consists of two icon packs, based on Font-Awesome 7.1.0

The Font-Awesome Icon Pack allows to use most of the free Font-Awesome icons as note icons. It uses the `fa-solid` prefix.
<img width="637" height="644" alt="image" src="https://github.com/user-attachments/assets/1346d494-94c1-48bf-b6ae-486c0e74921e" />

The Font-Awesome Brands Icon Pack allows to use most of the free Font-Awesome brand icons as note icons. It uses the `fa-brands` prefix.
<img width="631" height="639" alt="image" src="https://github.com/user-attachments/assets/814db35e-2857-4837-af42-e4c364feae4c" />

The fonts included are the [Font-Awesome WebFonts](https://github.com/FortAwesome/Font-Awesome/tree/7.x/webfonts), distributed by Fonticons under the [SIL Open Font License](https://github.com/FortAwesome/Font-Awesome?tab=License-1-ov-file#readme).
